# حاسبة تكلفة تصدير الفاكهة (مصر ← العراق) — تطبيق فلاتر

نسخة تطبيق موبايل (Flutter/Dart) من نفس حاسبة موقع PHP المرفقة، بنفس الحقول
والقيم الافتراضية ونفس معادلات الحساب الموجودة في `assets/js/app.js`، إضافة إلى
جلب سعر الدولار/الجنيه تلقائيًا من مصدرين عامين (بنفس فكرة `api/rate.php`) مع
سعر احتياطي إذا تعذّر الاتصال.

## ⚠️ ملاحظة مهمة جدًا حول ملف .ipa

ملف `.ipa` لا يمكن إنتاجه إلا عبر **Xcode على جهاز Mac**، مع **حساب Apple
Developer** لتوقيع التطبيق (Code Signing) — هذا قيد من Apple نفسها وليس قيدًا
في الكود. بيئة العمل التي أنشأت فيها هذا المشروع هي Linux بدون أي إمكانية
لتشغيل Xcode، لذلك **لا يمكن تسليمك ملف .ipa جاهزًا مباشرة**.

ما قمت به بدلًا من ذلك: أنشأت لك **مشروع فلاتر كامل وجاهز للبناء**
(كل شيفرة Dart مكتوبة وصحيحة). تبقى خطوة واحدة فقط للحصول على `.ipa`، ولديك
3 طرق لإتمامها:

### الطريقة 1 — إن كان لديك جهاز Mac
```bash
flutter create --org com.yourcompany fruit_export_calculator_tmp
# انسخ مجلد lib/ وملف pubspec.yaml من هذا المشروع إلى المشروع الجديد (استبدال)
cd fruit_export_calculator_tmp
flutter pub get
flutter build ipa --release
# أو لاختبار سريع على آيفون متصل بالكيبل:
flutter run --release
```
الناتج سيكون في: `build/ios/ipa/*.ipa`
(يتطلب فتح `ios/Runner.xcworkspace` مرة واحدة في Xcode لضبط Signing & Team
الخاص بحسابك في Apple Developer).

### الطريقة 2 — بدون جهاز Mac: خدمة بناء سحابية (الأسهل)
استخدم خدمة مثل **Codemagic** أو **GitHub Actions (macos runner)** أو
**Ionic Appflow**، وكلها تتيح بناء تطبيقات iOS من السحابة دون أمتلاك Mac:
1. ارفع هذا المشروع إلى مستودع GitHub/GitLab.
2. اربطه بـ Codemagic (له خطة مجانية محدودة) واختر "Flutter" كنوع المشروع.
3. أضف شهادة وملف Provisioning Profile من حساب Apple Developer الخاص بك
   (Codemagic يشرح ذلك في لوحة التحكم).
4. شغّل البناء، وستحصل على ملف `.ipa` جاهز للتنزيل.

### الطريقة 3 — تجربة سريعة بدون بناء iOS
لتجربة الحاسبة فورًا على أي جهاز (أندرويد، أو متصفح، أو حتى محاكي iOS إن كنت
على Mac) بينما تجهّز التوقيع:
```bash
flutter create --org com.yourcompany fruit_export_calculator_tmp
# استبدل lib/ و pubspec.yaml كما في الطريقة 1
flutter pub get
flutter run
```

## بنية المشروع
```

  main.dart                     # نقطة الدخول + الثيم + اتجاه RTL
  screens/calculator_screen.dart# الواجهة الكاملة + منطق الحساب
```

## القيم الافتراضية (مطابقة للصفحة الأصلية)
- وزن البراد: 30,000 كغم
- أجرة التوصيل داخل مصر: 6,000 دولار
- التخليص الجمركي: 1,600 دولار
- توصيل العراق: 1,000 دولار
- سعر الصرف العراقي الافتراضي عند فتح التطبيق: 138,000 د.ع لكل 100 دولار
  (وعند الضغط على "إعادة القيم الافتراضية" يصبح 155,000، تمامًا كما في
  زر `resetBtn` في الموقع الأصلي)
- وزن الصندوق: 15 كغم
- سعر احتياطي للدولار إن تعذّر الاتصال: 48.50 ج.م

## تحسين بسيط أضفته
الصفحة الأصلية كانت تحسب "سعر الصندوق بالدينار العراقي" في `app.js` لكنها لا
تعرضه فعليًا في الواجهة (لا يوجد عنصر له في `index.php`). أضفته كبطاقة نتيجة
سادسة في التطبيق لأنه رقم مفيد عمليًا.

## تغيير الألوان أو الشعار
الألوان مأخوذة من `assets/css/style.css` الأصلي (تركواز #0F766E). لتغييرها
عدّل القيم الثابتة في أعلى ملف `calculator_screen.dart`. لإضافة أيقونة
التطبيق استخدم حزمة `flutter_launcher_icons` بعد توليد مشروع iOS كامل.
